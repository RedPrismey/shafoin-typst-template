Template de rapport customisés réalisés avec [Typst](https://typst.app/).

## Customisation

Nécessite les polices :

- Metropolis
- Stretch Pro

4 thèmes disponibles :

- Bleu (blue-theme) (défaut)
- Rouge (red-theme)
- Vert (green-theme)
- Pastel (pastel-theme)
